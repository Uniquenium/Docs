# UniDeskComboBox 类型
下拉菜单

| 控件类型    | 元件                                                                              |
| ------- | ------------------------------------------------------------------------------- |
| 源代码文件路径 | `UniDesk/UniDeskComboBox.qml`                                                   |
| 继承      | [QtQuick ComboBox](https://doc.qt.io/qt-6.8/qml-qtquick-controls-combobox.html) |

## 属性
### `property bool enableFontDelegate`
启用后，每个菜单选项将根据文字内容展示不同的字体
