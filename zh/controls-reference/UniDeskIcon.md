# UniDeskIcon 类型
可修改颜色图标

| 控件类型    | 元件                                                               |
| ------- | ---------------------------------------------------------------- |
| 源代码文件路径 | `UniDesk/UniDeskIcon.qml`                                        |
| 继承      | [QtQuick Image](https://doc.qt.io/qt-6.8/qml-qtquick-image.html) |
### `property string iconSource`
图标所用图片的源
### `property color iconColor`
图标的颜色
### `property double iconSize`
图标的大小
